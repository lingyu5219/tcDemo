
#职位管理URL ——李逢杰

1.进入职位管理功能：hrmgt/positionList

2.查询职位列表：hrmgt/queryPositionList

3.打开新增职位页面：forward?page=hrmgt/positionAdd

4.新增职位：hrmgt/addPosition

5.打开修改职位页面：forward?page=hrmgt/positionModify

6.修改职位：hrmgt/updatePosition

7.删除职位：hrmgt/delPosition

8.打开查看职位页面：forward?page=hrmgt/positionDetail
