#系统公告&系统参数&签到URL－董皓
1.进入系统公告管理功能：notice/noticeList
2.进入系统参数管理功能：parmeter/parmeterList
3.查询系统公告列表：notice/queryNoticeByTitle
4.查询系统参数列表：parmeter/queryParmeter
5.打开新增系统公告页面：forward?page=system/insertNoticeUI
6.打开新增系统参数页面：forward?page=system/insertParmeterUI
7.新增系统公告：notice/insert
8.新增系统参数：parmeter/insert
9.打开修改系统公告页面：forward?page=system/updateNoticeUI
10.打开修改系统参数页面：forward?page=system/updateParmeterUI
11.修改系统公告：notice/update
12.修改系统参数：parmeter/update
13.删除系统公告：notice/delete
14.删除系统参数：parmeter/delete
15.打开管理员查看系统公告页面：forward?page=system/noticeDetail
16.进入非管理员查看系统公告列表：forward?page=system/noticeListToSTU
17.打开所有人查看系统公告页面：forward?page=system/noticeDetail
18.打开查看系统参数页面：forward?page=system/parmeterDetail
19.签到：sign/in
