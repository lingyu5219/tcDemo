#学生管理URL ——陈镇秋

1.进入学生管理功能：student/studentList

2.查询学生信息：student/queryStudentList 

3.打开新增学生信息页面：forward?page=stumgt/StudentAdd 

4.删除学生信息：student/deleteStudentById 
 
5.修改学生信息：student/updateStudent 

6.打开学生信息修改页面：forward?page=stumgt/studentModify

7.打开学生信息展示页面：forward?page=stumgt/studentDetail
