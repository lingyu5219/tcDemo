#日历管理URL ——钱兆瑞

1.进入日历页面：system/calendarList

2.查询日历列表：system/queryCalendarList

3.打开新加日历界面：forward?page=system/calendarAdd

4.新增日历：system/addCalendar

5.打开修改日历界面：forward?page=system/calendarAdd

6.修改日历：system/updateCalendar

7.删除日历：system/delCalendar

8.打开新加日历界面：forward?page=system/calendarDetail
